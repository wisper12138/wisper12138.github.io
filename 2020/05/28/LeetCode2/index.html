6.字符串压缩

思路：可以用数组存压缩后的字符串，遍历比较相邻的字符，相同则计数器加1，不同则将当前计数器存到数组，继续比较下一个。

StringBuilder: 前后两个比较，相同则加1

class Solution {
    public String compressString(String S) {
        //如果长度为1或空肯定返回原字符串
        if (S.length()<=1)
            return S;
        StringBuilder sb = new StringBuilder().append(S.charAt(0));

        int j=1;
        for (int i=1;i<S.length();i++){
            if (S.charAt(i)==S.charAt(i-1))
                j++;
            else{
                sb.append(j).append(S.charAt(i));

                j=1;
            }
        }
        return sb.append(j).length()<S.length()?sb.toString():S;
    }
}

双指针 找到每个字符之后第一个不相同的位置，一次性计数j-i

class Solution {
    public String compressString(String S){
            int N=S.length();
            int i=0;
            StringBuilder sb=new StringBuilder();
            while(i<N){
            int j=i;
            while(j<N &&S.charAt(j)==S.charAt(i)){
            j++;
            }
            sb.append(S.charAt(i));
            sb.append(j-i);
            i=j;
            }
        return sb.length()<S.length()?sb.toString():S;
    }
}

7.旋转矩阵

思路：这种题通常可以找到下标的规律。只要观察原下标的位置换成了什么下标就可以发现，原下标(i,j)换成了(n-1-j,i),并且换到了(j,n-1-i)的位置上，四个位置的元素满足这个交换规律为一组，所以只需要遍历nn/4个元素进行交换就可以了，以4-4矩阵为例，下标位置为 *，**标的位置

class Solution {
    public void rotate(int[][] matrix) {
        int l=matrix.length;
        for (int i=0;i<l-1;i++){
            for (int j=0;j<=i&&j<l-1-i;j++){
                int t=matrix[i][j];
                matrix[i][j]=matrix[l-1-j][i];
                matrix[l-1-j][i]=matrix[l-1-i][l-1-j];
                matrix[l-1-i][l-1-j]=matrix[j][l-1-i];
                matrix[j][l-1-i]=t;
            }
        }
    }
}
  • 换两次，先转置再中线对换

原矩阵为：
[1,2,3]
[4,5,6]
[7,8,9]

转置之后，即按斜对角线进行数据交换后的结果为：
[1,4,7]
[2,5,8]
[3,6,9]

矩阵再根据中间列进行对折，即中间列两边元素进行交换，得到：
[7,4,1]
[8,5,2]
[9,6,3]

class Solution {
    public void rotate(int[][] matrix) {
        int n = matrix.length;
        //矩阵转置，即按斜对角线进行数据交换
        for(int i=0;i<n;i++){
            for(int j=i;j<n;j++){
                int temp = matrix[i][j];
                matrix[i][j] = matrix[j][i];
                matrix[j][i] = temp;
            }
        }

        //矩阵根据中间列进行对折，得到最终结果
        for(int i=0;i<n;i++){
            for(int j=0;j<n/2;j++){
                int temp = matrix[i][j];
                matrix[i][j] = matrix[i][n-j-1];
                matrix[i][n-j-1] = temp;
            }
        }
    }
}

8.零矩阵

思路：直接找，找到然后遍历行列设为0肯定复杂度太大。可以第一遍遍历先找到0，然后用两个个集合分别记录下i和j，第二遍遍历再将在记录里的下标设为0。

class Solution {
    public void setZeroes(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        HashSet set1 = new HashSet<>();
        //HashSet set2 = new HashSet<>();用一个集合就可以
        for (int i=0;i<m;i++){
            for (int j=0; j<n;j++){
                if (matrix[i][j]==0){
                    set1.add(i);
                    set1.add(j+m);
                }
            }
        }
        for (int i=0;i<m;i++){
            for (int j=0;j<n;j++){
                if (set1.contains(i)||set1.contains(j+m)){
                    matrix[i][j]=0;
                }
            }
        }
    }
}

上面方法用集合运行太慢。换成布尔数组快2ms,想法是一样的。

class Solution {
    public void setZeroes(int[][] matrix) {
        boolean[] line = new boolean[matrix.length];
        boolean[] column = new boolean[matrix[0].length];
        // 找出要清零的行列
        for (int i = 0; i < matrix.length; i++) {
            for (int j = 0; j < matrix[0].length; j++) {
                if (matrix[i][j] == 0) {
                    line[i] = true;
                    column[j] = true;
                }
            }
        }
        // 开始对行清零
        for (int i = 0; i < matrix.length; i++) {
            if (line[i]) {
                for (int j = 0; j < matrix[0].length; j++) {
                    matrix[i][j] = 0;
                }
            }
        }

        // 开始对列清零
        for (int i = 0; i < matrix[0].length; i++) {
            if (column[i]) {
                for (int j = 0; j < matrix.length; j++) {
                    matrix[j][i] = 0;
                }
            }
        }

    }
}

9.字符串轮转

思路：首先两个字符串长度不同直接返回false，两个都是空串返回true。符合题目特点的s1和s2是每个字符串被分成两个子串，只要这两个子串分别相同则true。遍历s1，找到和s2第一个元素相同的位置，从这个位置分成两个子串分别比较。

class Solution {
    public boolean isFlipedString(String s1, String s2) {
        if (s1.length()!=s2.length()) {
            return false;
        }
        if (s1.length()==0)
            return true;
        int l= s1.length();
        for (int i=0;i<l;i++){
            if (s1.charAt(i) ==s2.charAt(0)){
                if (s1.substring(i,l).equals(s2.substring(0,l-i))&&
                    s1.substring(0,i).equals(s2.substring(l-i,l))){
                    return true;
                }
            }
        }
        return false;
    }
}
  • 只调用一次检查子串方法，运行更快：

将s1与s1拼接成一个新字符串，则如果符合题意，s2必定是新串的子串。

class Solution {
    public boolean isFlipedString(String s1, String s2) {
        if (s1.length()!=s2.length())
            return false;
        return (s1+s1).contains(s2);

    }
}
